About this project

The lion of Addis,
finally in your pocket.

Anbessa City Bus has served Addis Ababa since 1945 — over 80 years moving the city. With 1,000+ buses and nearly 300 million annual rides, it's the backbone of daily life for millions.

Yet navigating it is still done by word-of-mouth. I'm building the native Android app that changes that: live routes, arrival times, and the full network — in Amharic and English.

Anbessa
Navigator App

A native Android app mapping every Anbessa bus route across Addis Ababa — built entirely in Java with XML layouts. Shows estimated arrivals, guides riders door-to-door, and works in Amharic and English for the city's 1 million daily commuters who currently rely on guesswork.

How I'm building it

The build process

Pure native Android — Java and XML, built the right way from the ground up.

🗺️
01 — Research
Mapping the network
Riding Anbessa buses to collect real route data, stop coordinates, and schedule patterns across Addis Ababa.
Field research
🎨
02 — Design
XML Layouts & UI
Designing every screen in XML — ConstraintLayout, RecyclerViews, and custom views styled in Anbessa red and yellow.
Java · XML · Material Design
☕
03 — Build
Java + Android SDK
Activities, Fragments, ViewModels, and LiveData architecture. Google Maps SDK for route overlays. Retrofit for API calls. Room DB for offline caching.
Java · Retrofit · Room · Firebase
🚀
04 — Launch
Play Store Pilot
Starting with 10 core routes, beta testing with Addis commuters, then expanding to the full Anbessa network.
Google Play · APK testing
